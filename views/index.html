Hybrid Design

Hybrid Board Game Components

FlowMaster is built on 7 core principles that define a true hybrid board game experience.

🎲

Physical Components

✓

A physical game board with player tokens, a deck of QR-coded question cards, and dice determine movement and turn order across the board.

💻

Digital Component

✓

A full-stack web application handles user accounts, question delivery, answer validation, scoring, and live leaderboards across all players' devices.

🔗

Interdependence

✓

Neither component works alone — physical cards are meaningless without the web app to load them, and the digital game cannot progress without players drawing cards from the board.

📱

Digital Interaction

✓

Scanning the QR code on a physical card triggers the web app to load that specific question, directly linking a physical action to a digital response in real time.

📡

Data Sharing

✓

Points earned digitally are broadcast via WebSocket to all players, who then use those scores to move their physical tokens the corresponding number of spaces on the board.

📊

State Tracking

✓

The server tracks every player's score, current round, and game status. This digital state directly determines how far each player's token advances on the physical board.

⚡

Real-Time Feedback

✓

Answers, results, and leaderboard updates are pushed instantly to all connected devices via Socket.io. Players see scores change the moment a question is answered, keeping the physical and digital game perfectly in sync.
